# Frontend-mini-proyecto

Frontend estatico en JavaScript consumiendo el backend Spring.

## Conexion esperada

- Frontend: `http://localhost:5500`
- API backend: `http://localhost:8084`

## Arranque

Desde la raiz del repo frontend:

```powershell
node server.js
```

## Arranque con Docker

Desde la raiz del repo frontend:

```powershell
docker compose up --build -d
```
